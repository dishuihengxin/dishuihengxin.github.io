kubernetes简介

kubernetes是什么？

Kubernetes(k8s) 是 Google 基于 Borg 系统开源的一个容器分布式集群编排引擎项目，已经捐献给 CNCF(Cloud Native Computing Foundation) 社区管理，并且是 CNCF 第一个毕业的项目。Kubernetes 不仅仅是一个容器编排管理系统，更是提供一个应用集群架构的规范，定义应用服务的运行状态，Kubernetes 并自动通过内部调度使应用服务保持这种既定的状态。Kubernetes 作为云原生应用的基石，相当于一个云操作系统，重要性不言而喻。

kubernetes 为什么被大家简称为 k8s 呢？原来 kubernetes 起源于希腊语，寓意为舵手、领航员，8数字为ubernete中间字母的个数，这就西方常见的缩写命名方式。kubernetes

master 节点需要的端口号：

Protocol Direction Port Range Purpose Used By
TCP 入站 6443 (可自定义端口) api server all
TCP 入站 2379~2380 Etcd server client Api kube-apiserver,etcd
TCP 入站 10250 kubelet API self,control plane
TCP 入站 10251 kube-scheduler self
TCP 入站 10252 kube-controller-manager self

Node 节点需要的端口号：

Protocol Direction Port Range Purpose Used By
TCP 入站 10252 kubelet api self,control plane
TCP 入站 30000~32767 nodeport service all
